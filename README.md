# 👨‍🦯 AI Navigation System for Visually Impaired People

## 🚀 Project Overview

An AI-powered assistive navigation system designed to help visually impaired individuals navigate safely using real-time object detection, OCR, GPS guidance and voice feedback.

---

## 🎯 Features

✅ Real-time Object Detection using YOLOv5

✅ OCR-based Text Recognition

✅ GPS Navigation Assistance

✅ Voice Guidance using Text-to-Speech

✅ Obstacle Detection and Alerts

---

## 🛠️ Technologies Used

- Python
- YOLOv5
- OpenCV
- OCR
- Text-to-Speech
- Computer Vision
- Machine Learning
